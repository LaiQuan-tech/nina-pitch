# nina-pitch

印刷檔線上收稿系統 — **提案簡報**（Apple 風全螢幕 pitch deck，模仿 `LaiQuan-tech/agec-pitch`）。

- 線上：<https://nina-pitch.vercel.app>
- 純靜態站（單一 `index.html`）。
- **push 到 `main` → Vercel 自動部署**（Vercel 專案 `nina-pitch`，framework=static，原生 GitHub 整合）。

## 內容
封面 → 為什麼 → 四大痛點 → 解法 → 核心功能 → 特色三張（檔名把關／自動工單／管理後台，含 UI mockup）→ 運作流程 → 實機預覽（內嵌 nina-navy）→ 一個月時程 → 報價（未稅 58,000／含稅 60,900）→ 結尾。
